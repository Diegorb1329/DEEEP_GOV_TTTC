    gitcoin is bundled with several user identities from GitHub account, twitter, to wallet address
Which make it a potential candidate to become the valid single sign-on provider as facebook or google.

For the short/mid term, it will be nice if the user can login with his/her wallet address within the gitcoin ecosystem.

Benefits:

User can use their wallet (metamask or web connect) to login to gitcoin and the discord server (https://gov.gitcoin.co/)

Possible approach: 

integrate the eauth server with gitcoin backend, ex:
